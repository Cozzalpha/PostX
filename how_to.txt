A Django-based automation platform that uses **Google Gemini** to generate content and **Celery/Redis** to schedule and publish posts directly to **Instagram/Facebook**.

## üìã Prerequisites

Before running the project, you must have these installed:

1.  **Python 3.10+**: [Download Here](https://www.python.org/downloads/)
2.  **Docker Desktop**: [Download Here](https://www.docker.com/products/docker-desktop/) (Required for Redis)
3.  **Ngrok**: [Download Here](https://ngrok.com/download) (Required for public image URLs)

---

## üõ†Ô∏è Installation & Setup

### 1. Database & Project Setup

1.  **Clone the repository:**
    ```bash
    cd social_agent
    ```

2.  **Create and Activate Virtual Environment:**
    * **Windows:**
        ```bash
        python -m venv venv
        venv\Scripts\activate
        ```
    * **Mac/Linux:**
        ```bash
        python3 -m venv venv
        source venv/bin/activate
        ```

3.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Initialize the Database:**
    ```bash
    python manage.py makemigrations
    python manage.py migrate
    ```

5.  **Create an Admin User:**
    ```bash
    python manage.py createsuperuser
    ```

### 2. Ngrok Setup (Crucial for Facebook API)

Facebook cannot access images on `localhost`. You must tunnel your local server to the internet.

1.  **Install Ngrok** from the link above.
2.  **Authenticate Ngrok** (Run this once):
    ```bash
    ngrok config add-authtoken YOUR_NGROK_TOKEN
    ```
    *(Get your token from the [Ngrok Dashboard](https://dashboard.ngrok.com/get-started/your-authtoken))*
